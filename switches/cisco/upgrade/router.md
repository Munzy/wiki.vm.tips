# Upgrade Cisco Router IOS

## Download

https://software.cisco.com/download/home

Search for the appropriate IOS software.

## Upgrade Steps

### Connect

Connect into the router using either serial, telnet, or ssh.

### Enable

```
enable
```

### Filesystem

```
show file systems
```

### Copy

```
copy tftp: flash:

```
