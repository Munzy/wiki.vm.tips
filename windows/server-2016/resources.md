# Minimum Requirements


|  Resource |      Minumum      |
|:---------:|:-----------------:|
| Processor | 1.4 GHz 64 Bit    |
| Memory    | 512 MB / 2 GB ECC |
| Storage   | 32 GB             |
| Network   | 1 Gigabit         |

|:---------:|:--------------:|:--------------:|
| Processor | 240 Processors | 512 Processors |
|   Memory  |      12 TB     |      24 TB     |
